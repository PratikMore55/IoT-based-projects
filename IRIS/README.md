# IRIS

IRIS is a Smart device for a blind indivisuals which serves various functionalities powered by Raspberry Pi, featuring Pi Camera and Neo6M module for enhanced functionality and navigation.

---
## REQUIREMENTS:
---

### HARDWARE:

- Raspberry Pi
- Pi Camera Module
- Neo-6m Module
- Buttons
- Earphone

### SOFTWARE

- VNC Viewer
- OCR-GTTS
- OpenCV
- Telegram Bot API
