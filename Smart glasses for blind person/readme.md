# SMART GLASSES FOR BLIND PEOPLE

This IoT project basically works as a third eye for visually impaired people.

# COMPONENTS USED :-

## Hardware :-
- Arduino UNO  
- Ultrasonic sensor (HC SR04) 
- MP3-TF-16P 
- 0.5W 8OHM trumpet 
- Power supply 
- Jumper wires  

## Software :-

- Arduino IDE
